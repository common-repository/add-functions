With this plugin you can add new functions through admin menu, without editing your themes functions.php

== Description ==
Easy way to add php code. No need to edit functions.php in your theme directory.Just paste code in Admin menu and save.Code is highlighted with <a href="http://codepress.sourceforge.net/index.php">codepress</a>.
== Installation ==
1. Upload '/add-functions/' to the '/wp-content/plugins/' directory2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add code in Add Functions menu in Options
